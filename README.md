# Calculadora em Java (Swing)

Este é um projeto simples de uma calculadora com interface gráfica (GUI) desenvolvida em Java utilizando a biblioteca Swing. O projeto foi construído para realizar operações matemáticas básicas.

## 🚀 Funcionalidades

- **Adição (+)**
- **Subtração (-)**
- **Multiplicação (*)**
- **Divisão (/)**
- Interface gráfica interativa construída com `javax.swing`.
- Tratamento de erros básicos com timer (limpa a tela de erro automaticamente).

## 🛠️ Tecnologias Utilizadas

- **Java**
- **Java Swing** (para a interface gráfica)
- **NetBeans IDE** (geração e design do Form)

## 📁 Estrutura do Projeto

O código-fonte está dividido em dois pacotes principais:
- `operations`: Contém a classe `Operations.java` responsável pela lógica matemática (soma, subtração, multiplicação e divisão).
- `br.udf.edu`: Contém a classe principal `Calc.java` que gerencia a interface gráfica e captura as interações do usuário.

## ⚙️ Como Executar

1. Certifique-se de ter o **Java Development Kit (JDK)** instalado em sua máquina.
2. Clone este repositório:
   ```bash
   git clone https://github.com/SEU_USUARIO/NOME_DO_REPOSITORIO.git
   ```
3. Abra o projeto na sua IDE favorita (NetBeans, Eclipse, IntelliJ, etc).
4. Execute a classe `Calc.java` (localizada em `src/br/udf/edu/Calc.java`).
